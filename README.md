# RoadCoin

*Drive the road. Earn the fuel.*

Native token economy that rewards every real action on The BlackRoad. Not speculation — real fuel earned through real work.

## The Ride

Every mile earns RoadCoin. Drive the road, earn the fuel. Solve homework? Earn. Create content? Earn. Help someone? Earn. The more you ride, the more the road gives back.

## What It Does

Closed-loop token economy where every meaningful action across all 14 products automatically earns ROAD. Spend it on premium compute, agent unlocks, and boosted reach. Backed by RoadChain, bridgeable to Base L2 via Coinbase.

## Integrations

| Service | Role |
|---------|------|
| **Cloudflare Workers** | Token economy runtime |
| **Cloudflare D1** | Balances, transactions, earn/spend ledger |
| **Coinbase Commerce** | Fiat purchase — card, Apple Pay, crypto |
| **Coinbase Onramp** | USDC/ETH on Base — zero-auth deposits |
| **Basescan API** | Public deposit detection on Base L2 |
| **Stripe** | Fiat purchase via traditional payment rails |
| **RoadChain** | Every transaction hashed on the ledger |
| **Ollama** | Per-inference minting from Pi fleet (0.0025 ROAD/run) |
